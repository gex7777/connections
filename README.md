# connections
App to view degree of separation between two people
# Features
1. Add User
  - Users can be added using the Floating Action Button
  - The popup form has builtin validations
  - Assigns the user a unique Avatar
2. Edit/Delete User
  - Once the user is added, they can be Edited or deleted using the card action dropdown on the top right
3. View Degree of Seperation
  - when two user cards are selected, a button appears to view the degree of seperation
  - on clicking, a popup show the degree of seperation
